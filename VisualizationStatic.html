One of the strengths of R is the wealth of data plotting packages. This CRAN Task View maintains a list of static graphics packages, and links to web resources on plotting data.

Note that some of these packages are on CRAN and others are on GitHub, Bioconductor, or R-Forge.

If you think that a package is missing from this list, please let us know through issues or pull requests in the GitHub repository.

The underlying grid system

  • grid | Implements the primitive graphical functions.
  • gtable
  • graphics (base) | The original graphics engine.
  • gridExtra
  • gridGraphics | Redraw Base Graphics Using ‘grid’ Graphics.

Lattice graphics

  • lattice | Create data visualisations using the trellis method. Built on the grid system.
  • latticeExtra | Extra graphical utilities based on lattice.
  • adegraphics | Lattice-based package for the representation of multivariate data.
  • loa | Lattice options and add-ons.
  • stripless | Structured trellis displays without strips for lattice graphics.
  • tactile | New and extended plots, methods, and panel functions for lattice
